# Installation

Install the Afrimomo SDK using your preferred package manager.

## npm

```bash
npm install afrimomo-sdk
```

## pnpm

```bash
pnpm add afrimomo-sdk
```

## yarn

```bash
yarn add afrimomo-sdk
```

## Requirements

- Node.js 18.0.0 or higher
- TypeScript 5.0+ (optional, but recommended)

## Package Contents

The SDK includes:

- Full TypeScript type definitions
- ESM and CommonJS builds
- Zero runtime dependencies for the core SDK

## Verify Installation

```typescript
import { AfromomoSDK } from "afrimomo-sdk";

console.log("Afrimomo SDK installed successfully!");
```
